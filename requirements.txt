В данном проекте использовались следующие библиотеки:

1. PyGame  
   Библиотека для разработки игр и мультимедиа приложений на Python. Включает поддержку графики, звука, ввода и т.д.

2. TKinter  
   Стандартная библиотека Python для создания графического интерфейса пользователя (GUI). Основывается на Tcl/Tk.

3. PIL (Python Imaging Library)  
   Расширение для работы с изображениями в Python. Используется совместно с Tkinter для поддержки отображения изображений в GUI.

4. os  
   Модуль стандартной библиотеки Python для работы с файловой системой и операционной системой.
Эти четыре пакета являются основными зависимостями проекта.
